# Generated Logbook 500

This folder contains 500 synthetic Leaf logbook JSON files for testing web app performance with a larger SD-card logbook.

Copy the contents of `logbook/` to the Leaf SD card `/logbook` directory, or copy the whole `logbook` folder if the SD card does not already have one.

The files are intentionally reasonable-looking but synthetic:

- one JSON file per flight
- varied dates, durations, altitudes, climb/sink rates, distances, temperatures, pilots, and gliders
- occasional `metrics.max_wind` values
- no linked IGC/KML track files

Regenerate or delete this folder freely; it is test data only.
